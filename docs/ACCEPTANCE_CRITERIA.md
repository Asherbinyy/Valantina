# Acceptance Criteria — “Done means done”

## Must-have
- [ ] Boots and loads all assets without errors
- [ ] Start screen shows correctly (fonts, colors, CTA)
- [ ] Tap/click jump works reliably
- [ ] Player auto-runs at a stable speed
- [ ] Collision hazards trigger bonk + checkpoint reset (no game over)
- [ ] Hearts collectible with pop + SFX
- [ ] At least 2 micro-notes appear at correct moments
- [ ] Pause overlay works: Resume + Restart + Sound toggle
- [ ] Finish trigger launches cutscene then Valentine prompt
- [ ] YES triggers confetti + final message + coupon
- [ ] Restart works from Pause, Prompt, Celebration (always)
- [ ] Works on mobile & desktop web
- [ ] Credits file exists and lists all external assets

## Nice-to-have (only if time)
- [ ] Parallax background
- [ ] Subtle camera shake on bonk
- [ ] Landing squash
- [ ] Screenshot hint on final screen
